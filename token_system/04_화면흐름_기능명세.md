# 화면 흐름 및 기능 명세서
> 특수학급 화폐·금융 시스템  
> 최종 정리: 2026-03-21 (v3 → v4 번호 정규화)
> 변경 이력: v3 — 학급 개설·로그인 흐름, 커스텀 주식 관리, 동의서 템플릿 화면 추가

---

## 1. 접근·인증 흐름

### 1.1 최초 학급 개설 (교사)
```
landing page 접속
    ↓
"새 학급 개설" 클릭
    ↓
학교명 / 학급명 / 교사명 / 교사 PIN 4자리 입력
    ↓
시스템이 학급 식별코드 자동 발급 (예: NDG-2026-001)
    ↓
대시보드 진입 + 코드 안내 화면
    ("이 코드를 메모해 두세요. 이후 로그인에 사용됩니다.")
```

### 1.2 교사 로그인
```
학급 식별코드 입력
    ↓
교사 PIN 4자리 입력
    ↓
교사 대시보드 진입
```

### 1.3 학생 로그인 (개인 태블릿)
```
학급 식별코드 입력 (또는 QR 스캔)
    ↓
학생 이름 목록에서 선택
    ↓
PIN 4자리 입력
    ↓
학생 모바일뱅킹 홈 진입
```

### 1.4 ATM 접근 (공용 태블릿)
```
QR카드 스캔
    ↓ (PIN 없이 즉시)
잔액 확인 + 출금 화면
```

## 2. 교사 PC 화면 전체 목록

### 2.1 대시보드 (메인)
- 학생별 현재 잔액 카드 목록
- 오늘 PBS 체크 현황 (체크됨/미체크)
- DRO 타이머 실시간 현황
- 소거 경고 알림 배지
- 오늘의 자동 정산 예정 금액
- 커스텀 주식 현황 요약

### 2.2 PBS 체크 화면 ⭐ 핵심
- 학생 선택 탭
- 활성 PBS 목표 목록 + 발생 횟수 입력
- 촉구 수준 선택
- 셀프 체크 승인/거절
- DRO 타이머 시작/리셋
- 상황 메모 입력

### 2.3 학생 관리
- 학생 목록 + QR코드 출력
- PIN 초기화 기능
- PBS 지원 단계 설정
- 반응대가 활성화 스위치 (보호자 동의 체크 연동)

### 2.4 PBS 목표 관리
- 학생별 목표 행동 등록/수정/비활성화
- 전략 유형 태그 (DRA/DRI/DRO/FCT 등)
- 행동형성 단계 조정
- 월별 단가 조정 이력

### 2.5 가게 관리
- 아이템 등록/수정/비활성화
- 재고 현황 + 소진 알림
- 카테고리별 필터

### 2.6 주식 창구
**날씨 연동 탭:**
- 오늘 날씨 기반 주가 자동 표시
- 학생별 보유 현황
- 매수/매도 처리 (교사 대행)

**커스텀 종목 탭 ✅ 신규:**
- 커스텀 종목 목록 + 현재 주가
- 종목 추가 (이름 + 이모지 + 초기가)
- 주가 조정 UI:
  ```
  [직접 입력] [🚀 폭등] [📈 상승] [➡️ 보합] [📉 하락] [💥 폭락]
  ```
- 내부 메모 입력 (학생에게 비공개)
- 조정 이력 보기

### 2.7 정산 설정
- 기본급 금액 설정
- 정산 스케줄 확인
- 이자율·최소잔액 설정
- 수동 정산 실행

### 2.8 통장 출력
- 학생 선택 + 기간 선택
- 미리보기 + PDF 출력/인쇄
- 학기말 종합본 일괄 출력

### 2.9 행동계약서
- 신규 작성 (GPT 초안 생성 포함)
- 계약 목록 + 상태 관리
- 서명 완료 후 활성화
- PDF 출력
- 레벨업 계약 갱신 흐름

### 2.10 AI FBA 분석
- 행동 기술 자연어 입력
- 선행사건/후속결과 체크리스트
- GPT 분석 요청 + 결과 표시
- 계약서 초안 연계

### 2.11 소거 알림
- 감지된 소거 상황 목록
- 위험도 표시 + GPT 권장 사항
- 교사 확인/처리

### 2.12 학급 공동계좌
- 공동 잔액 + 거래내역
- MVP 보너스 지급
- 학급 이벤트 사용

### 2.13 시스템 설정 ✅ 신규
```
학급 기본 설정
├── 화폐 단위 (기본: 500원)
├── 시작 잔액 (기본: 1,000원)
├── 최저잔액 보호 (고정: 500원)
├── 이자율 설정
├── 학기 이월 여부
├── 날씨 지역 설정
└── 데이터 보관 기간

학급 정보
├── 학교명 / 학급명
├── 학급 식별코드 확인
└── 학년도/학기 설정
```

### 2.14 동의서 관리 ✅ 신규
- 동의서 템플릿 목록 (유형별)
- 템플릿 미리보기 + 출력
- 동의서 유형:
  - 시스템 전반 동의서
  - 반응대가 적용 동의서
  - 행동계약 보호자 확인서
  - 데이터 보관 동의서

## 3. 학생 태블릿 화면

### 3.1 홈
- 잔액 크게 표시
- 오늘 획득 금액
- 주식 보유 요약
- 새 알림 (입금 알림)

### 3.2 통장 (거래내역)
- 날짜별 거래 목록
- 유형별 아이콘 (💰급여/🛒구매/🎁선물/📈주식)
- 주간·월간 필터

### 3.3 주식 시세
- 날씨 연동 4개 종목
- 커스텀 종목 (교사가 활성화한 것)
- 등락 표시 + 보유 현황
- "매수 요청" → 교사에게 알림

### 3.4 가게
- 아이템 목록 (재고 포함)
- 구매 + 선물하기

### 3.5 셀프 체크
- PBS 목표 중 셀프 체크 허용된 것
- 체크 요청 → 교사 승인 대기 표시

## 4. ATM 화면 (공용 태블릿)
```
1. 시작화면 → QR 스캔 안내
2. QR 인식 → 학생 확인 + 잔액 표시
3. 출금 → 금액 선택 → 확인
```

## 5. 자동화 스케줄 전체

| 시간 | 자동화 내용 |
|-----|-----------|
| 매일 오전 08:30 | 날씨 API → 날씨 연동 주가 자동 갱신 |
| 매일 오전 09:00 | 출석 기본급 자동 입금 |
| 매일 오후 03:00 | PBS 체크분 + 역할수행 일괄 정산 |
| 금요일 오후 02:00 | 주간 보너스 + DRL + 이자 계산 |
| 금요일 오후 02:30 | 주간 통장 자동 생성 |
| 계약 만료 D-7 | 갱신 알림 |
| 소거 감지 시 | 실시간 교사 알림 |
| 데이터 보관 만료 | 자동 삭제 또는 교사 알림 |
