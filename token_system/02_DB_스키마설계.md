## ⚠️ Schema 적용 필수 — SQL 실행 전 반드시 읽을 것

이 시스템의 **모든 테이블은 `pbs` schema에 생성**한다.
H.A.T.C.H. 대시보드와 동일 Supabase 프로젝트를 공유하며,
`public` schema와 충돌을 방지하기 위해 완전히 분리한다.

**SQL 실행 순서:**

```sql
-- 1. schema 생성 (최초 1회)
CREATE SCHEMA IF NOT EXISTS pbs;
GRANT USAGE ON SCHEMA pbs TO anon, authenticated;
ALTER DEFAULT PRIVILEGES IN SCHEMA pbs
  GRANT ALL ON TABLES TO anon, authenticated;

-- 2. 아래 각 테이블 앞에 pbs. 붙여서 실행
--    예: CREATE TABLE class_codes → CREATE TABLE pbs.class_codes

-- 3. behavior_db_seed.sql 실행 (Claude AI FBA 기능 필수)
```

**Supabase 클라이언트 설정 (반드시 schema 명시):**
```typescript
createBrowserClient(url, key, { db: { schema: 'pbs' } })
createServerClient(url, key, { cookies, db: { schema: 'pbs' } })
```

## 신규·수정 테이블 상세

### [인증] class_codes ✅ 신규
```sql
CREATE TABLE class_codes (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  code text UNIQUE NOT NULL,           -- 예: NDG-2026-001 (자동 생성)
  school_name text,                    -- 학교명
  class_name text,                     -- 학급명 (예: 남대구초 특수학급)
  teacher_name text,
  teacher_pin_hash text NOT NULL,      -- 교사 PIN bcrypt 해시
  academic_year int DEFAULT 2026,
  semester int DEFAULT 1,              -- 1학기/2학기
  is_active bool DEFAULT true,
  created_at timestamptz DEFAULT now(),
  expires_at timestamptz               -- 학년도 말 자동 만료
);
```

**식별코드 생성 규칙:**
```
형식: [학교약어]-[연도]-[순번]
예시: NDG-2026-001 (남대구초 2026년 1번째 학급)
생성: 학급 개설 시 서버에서 자동 발급
```

### [운영] students (수정)
```sql
CREATE TABLE students (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  class_code_id uuid REFERENCES class_codes(id),
  name text NOT NULL,
  grade int,
  pin_hash text NOT NULL,              -- PIN 4자리 bcrypt 해시 ✅ 추가
  qr_code text UNIQUE NOT NULL,        -- QR ID 카드값
  disability_type text[],
  pbs_stage int DEFAULT 1,
  behavior_function text,
  response_cost_enabled bool DEFAULT false,
  parental_consent_rc bool DEFAULT false,
  min_balance int DEFAULT 500,
  created_at timestamptz DEFAULT now(),
  is_active bool DEFAULT true
);
```

### [운영] system_settings ✅ 신규
```sql
-- 학급별 시스템 설정 (기본값 → 교사가 조정 가능)
CREATE TABLE system_settings (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  class_code_id uuid REFERENCES class_codes(id) UNIQUE,
  currency_unit int DEFAULT 500,         -- 화폐 단위 기본값
  starting_balance int DEFAULT 1000,     -- 학기 초 시작 잔액
  min_balance_protection int DEFAULT 500,-- 최저잔액 보호
  interest_rate_weekly numeric DEFAULT 0.005, -- 주간 이자율 (0.5%)
  interest_min_balance int DEFAULT 2000, -- 이자 지급 최소 잔액
  balance_carryover bool DEFAULT true,   -- 학기 이월 여부
  data_retention_months int DEFAULT 12,  -- 데이터 보관 기간 (개월)
  weather_location text DEFAULT '대구',  -- 날씨 API 지역
  updated_at timestamptz DEFAULT now()
);
```

### [운영] stock_prices (수정)
```sql
CREATE TABLE stock_prices (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  stock_name text NOT NULL,
  stock_type text DEFAULT 'weather',     -- 'weather' | 'custom' ✅ 추가
  price_date date NOT NULL,

  temperature_celsius numeric,
  precipitation_mm numeric,
  weather_condition text,
  calculation_note text,
  UNIQUE (stock_name, price_date)

```

### [운영] custom_stocks ✅ 신규
```sql
CREATE TABLE custom_stocks (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  class_code_id uuid REFERENCES class_codes(id),
  name text NOT NULL,                    -- 학생들이 직접 지은 이름
  emoji text DEFAULT '🎲',
  description text,
  current_price int NOT NULL DEFAULT 100,
  created_by text,                       -- 교사명
  named_by text,                         -- 종목 이름 지은 학생명
  is_active bool DEFAULT true,
  created_at timestamptz DEFAULT now()
);
```

### [운영] custom_stock_prices ✅ 신규
```sql
CREATE TABLE custom_stock_prices (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  stock_id uuid REFERENCES custom_stocks(id),
  price_date date NOT NULL,
  price int NOT NULL,
  previous_price int,
  adjustment_type text,
  -- 'manual_input'   교사 직접 입력
  -- 'surge'          폭등 버튼 (x2~x3)
  -- 'crash'          폭락 버튼 (x0.3~x0.5)
  -- 'flat'           보합 버튼
  teacher_note text,                     -- 내부 메모 (학생에게 비공개)
  created_at timestamptz DEFAULT now(),
  UNIQUE (stock_id, price_date)
);
```

**교사 주가 조정 UI 옵션:**
```
[직접 입력] 원하는 금액 입력
[폭등 🚀]   현재가 × 2~3배 (랜덤 또는 교사 선택)
[상승 📈]   현재가 × 1.2~1.5배
[보합 ➡️]   현재가 유지
[하락 📉]   현재가 × 0.7~0.8배
[폭락 💥]   현재가 × 0.3~0.5배
```

### [운영] consent_templates ✅ 신규

CREATE TABLE consent_templates (

## 인증 흐름별 접근 권한

```
교사 (class_codes + teacher_pin)
→ 전체 읽기/쓰기
→ 커스텀 주가 조정
→ 학생 PIN 초기화
→ 시스템 설정 변경

학생 (class_codes + student name + pin)
→ 본인 account, transactions 읽기
→ shop_items, stock_prices 읽기
→ 선물 송금 쓰기
→ pbs_records 셀프 체크 생성 (승인 대기)

ATM (QR 스캔)
→ 본인 계좌 잔액 읽기
→ 출금 처리 쓰기
```
